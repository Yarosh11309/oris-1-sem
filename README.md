**Описание:**  
Данный проект представляет собой клон сайта [ivi.ru](https://www.ivi.ru/). Проект включает в себя адаптивный дизайн, каталог фильмов, а также интуитивно понятный интерфейс для пользователей. Цель проекта — продемонстрировать навыки работы с современными веб-технологиями и создания масштабируемых, отзывчивых веб-приложений.

## Функциональные возможности

- **Главная страница:** Актуальные фильмы
- **Карточка фильма:** Подробное описание
- **Пользовательский интерфейс:** Удобная навигация
